## üìù Description

This interactive Power BI dashboard provides a comprehensive analysis of company sales performance across different regions, categories, and time periods. It visualizes key business metrics such as total sales, profit, and transaction counts, while also highlighting top-performing product subcategories and geographic sales distribution.

The dashboard enables business stakeholders to monitor performance trends, identify profitable products, and make data-driven decisions for improving regional and product-level sales strategies.

## üìÇ Data Source

The dashboard is based on a structured sales dataset (CSV or Excel format) containing:

Order Details: Order ID, Order Date, Sales, Profit, Quantity

Product Details: Category, Sub-Category

Customer & Region Info: State, Region

This dataset is inspired by the Superstore Sales Dataset, a popular dataset for Power BI and Tableau sales analytics.
